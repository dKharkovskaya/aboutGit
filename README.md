
# Работа с GIT

1. Команда `pwd` - выводит путь к текущей директории.
2. Команда `cd ~` - переход к домашней директории.
3. Команда `ls` - вывести содержимое директории.
4. Команда `cd имя_папки"` - сменить директорию.
5. Команда `cd ..` - подняться вверх на 1 директорию.
6. Команда `ls -a` - вывести содержимое директории вместе с скрытыми файлами
7. Команда `touch %ИМЯ_ФАЙЛА%` - создать файл.
8. Команда `mkdir %ИМЯ_ДИРЕКТОРИИ%` - создать директорию.
9. Команда `cp index.html src/` - скопировали index.html в папку src.
10. Команда `mv файл папка перемещения` - Синтаксис команды mv аналогичен синтаксису cp. После имени команды указывают список файлов и папок, которые нужно переместить, а затем — папку, в которую нужно выполнить перемещение.
11. Команда `cat %ИМЯ_ФАЙЛА%` - прочитать содержимое текстового файла.
12. Команда `rm %ИМЯ_ФАЙЛА%` - удалить файл.
13. Команда `rmdir %ИМЯ_ДИРЕКТОРИИ%` - удалить пустую директорию.
14. Команда `rm -r %ИМЯ_ДИРЕКТОРИИ%` - удалить директорию со всеми файлами внутри.

**При таком удалении файлы удаляются навсегда, не попадая в корзину**

**Команды в терминале необязательно вбивать и выполнять по очереди. Их можно указывать не по одной, а сразу списком. Для этого их нужно разделить двумя амперсандами (&&).**

15. Команда `git init` - делает папку, в которую сейчас находимся git репозиторием.
16. Команда `rm -rf .git` - приводит к удалению git репозитория в папке.
17. Команда `git status` - показывает текущее состояние репозитория.
18. Команда `git add --all` - добавляет все файлы для остлеживания.
19. Команда `git add %имя_файла%` - добавляет конкретный файл для остлеживания.
20. Команда `git add .` - добавляет всю текущую папку.
21. Команда `git commit -m "Мой первый коммит!"` - делает коммит папки с файлами
22. Команда `git log` - показать историю коммитов.
23. Команда `git clone %адрес удаленного репозиторию% ` - выкачивает проект с удаленного репозитория на локальную машину.
